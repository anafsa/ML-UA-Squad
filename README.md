# Create a virtual environment

```bash
# Create a virtual environment
python3.8 -m venv ml_ua_squad_env

# Activate a virtual environment
source ml_ua_squad_env/bin/activate

# Install the required libraries
pip install -r requirements.txt --no-cache-dir
```
